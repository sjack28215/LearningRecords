## 邻接矩阵
有向图 无向图

A[i][j] = 1 
```c++
#define MaxVertexNum 100
typedef char VertexType;
typedef int EdgeType;
typedef struct{
    // 一维数组存储点的信息
    VertexTpye vex[MaxVertexNum];
    // 二维数组存储边的信息
    // 邻接矩阵 空间复杂度O(n2)
    Edgetype Edge[MaxVertexNum][MaxVertexNum];
    // 非带权图
    // A[i][j] = 1 存在边 A[i][j] = 0不存在
    // 带权图
    // A[i][j] = w;A[i][j] = 0 | ∞
    int vexnum,edgenum;
}MGraph;
优点：容易判断顶点之间是否存在边 A[i][j]
缺点：统计边的数目(需要遍历)

适用：稠密图
```
