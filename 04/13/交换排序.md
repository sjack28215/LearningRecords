# 交换排序
基于两个元素的关键字的比较来进行排序的



## 冒泡排序

```c++

for(int i = 0; i < n-1; i++){
    for(int j = 0; j < n-1-i; j++){
        if(A[j] > A[j+1]){
            int temp = A[j];
            A[j] = A[j+1];
            A[j+1] = temp;
        }
    }
}
```

时间效率：最好：O(n) 最坏O(n<sup>2</sup>) 平均：O(n<sup>2</sup>)  
空间效率：O(1)
稳定性：稳定  
适用性：顺序存储 链式存储
