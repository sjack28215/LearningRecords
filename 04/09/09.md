- 无哨兵
每次循环都需要判断是否越界
比较次数：最多n次

```c++
int SequentialSearch(int a[],int n,int key){
    int i;
    for(int i = 0;i < n; i++){
        if(a[i] == key)
        {
            return i;
        }
    }
    return -1;
}
```
