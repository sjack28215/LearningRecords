# 交换排序
基于两个元素的关键字的比较来进行排序的



## 冒泡排序

```c++

for(int i = 0; i < n-1; i++){
    for(int j = 0; j < n-1-i; j++){
        if(A[j] > A[j+1]){
            int temp = A[j];
            A[j] = A[j+1];
            A[j+1] = temp;
        }
    }
}
```

时间效率：最好：O(n) 最坏O(n<sup>2</sup>) 平均：O(n<sup>2</sup>)  
空间效率：O(1)
稳定性：稳定  
适用性：顺序存储 链式存储

## 快速排序


基本思想：  
1.排序表任取一个元素pivot 
2.将元素表分为两部分一部分小于pivot 一部分大于pivot
33.递归两个子部分重复以上操作

```c++
int Partition(ElemType A[],int low,int high){
    ElemType pivot = A[low];
    while(low<high){
        while(low<high && A[high]>=pivot) --high;
        A[low] = A[high];
        while(low<high && A[low] <=pivot) ++low;
        A[high]=A[low];
    }
    A[low]=pivot;
    return low;
}
```
快排是递归的 所以要用栈来保存 容量与递归调用的最大层数一致
空间效率：O(log<sub>2</sub>n)  
时间效率：最坏 O(n<sup>2</sup>)  最好O(nlog<sub>2</sub>)  
稳定性：不稳定  
适用：顺序存储
