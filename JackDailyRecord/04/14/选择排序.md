## 选择排序
主要思路：每一趟选择最小的元素
##  简单选择排序

```c++
void SelectSort(ElemType A[],int n){
    for(int i = 0;i < n-1;i ++){
        int min = i;
        for(int j = i+1;j < n;j ++){
            if(A[j]<A[min]) min = j;
        }
        if(min!=i) swap(A[i],A[min]);
    }
}
```

空间效率：O(1)
时间效率：O(n<sup>2</sup>)  
稳定性：不稳定  
适用性：顺序存储和链式存储的线性表


## 堆排序 
堆 可看作 一颗完全二叉树
定义：
1. L(i) >= L(2i) L(i)>=L(2i+1)  大根堆
2. 或者L(i) <= L(2i) L(i)<=L(2i+1) 小根堆


思路：
1. 如何将无序序列构造成堆
2. 输出堆顶元素后 如何将剩余的元素调整成堆


```c++
void BuildMaxHeap(ElemType A[],int len){
	for(int i = len/2;i >0;i--){
		HeadAdjust(A,i,len);
	}
}

void HeadAdjust(ElemType A[],int k, int len){
	A[0] = A[k];
	for(int i = 2*k;i <= len;i *= 2){
		if(i < len && A[i]<A[i+1]){i++}
		if(A[0]>=A[i]) break;
		else{
		A[k]=A[i];
		k=i;
		}
	}
	A[k]=A[0];
}
```

时间复杂度：`O(n)`

堆排序算法
```c++
void HeapSort(ElemType A[],int len){
	BuildMaxHeap(A,len);
	for(int i = len;i > 1;i --){
		Swap(A[i],A[1]);
		HeadAdjust(A,1,i-1);
	}
}
```

空间效率：O(1)
时间效率：O(nlog<sub>2</sub>n)
稳定性：不稳定
适用性：顺序存储的线性表
