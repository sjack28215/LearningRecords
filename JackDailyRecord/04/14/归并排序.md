思路 ：将两个或者两个以上的有序表合成一个新的有序表
二路归并操作


```c++
ElemType *B =(ElemType*)malloc((n+1)*sizeof(ElemType));
void Merge(ElemType A[],int low,int mid,int high){

	int i,j,k;
	for(k = low;k <= high;k++){
		B[k]=A[k];
	}
	
	for(i = low,j=mid+1,k=i;i<=mid&&j<=high;k++){
		if(B[i]<=B[j]){
		A[k] = B[i++];
		}
		else{
		A[k] = B[j++];
		}
		while(i<=mid) A[k++] = B[i++];
		while(j<=high) A[k++] = B[j++];
	}
}
```
