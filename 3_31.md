## html区块
### 块元素
新行开始 独占一行
包含其他块元素和行内元素
```html
<div> <p> <h1> <ul> <ol> <table> <form>
```
//div标签用于创建块内容器 以便于组织页面的结构和布局

### 行内元素
只在同一行内呈现 不会独占一行
只占据其内容所需的宽度
行内元素不能包含块级元素 但可以包含其他行内元素
```html
<span> <a> <strong> <em> <img> <br> <input>
<span>用于内联式样式化文本
```
